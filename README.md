**🎮 Projet Call of Data
**


📊 Analyse de la satisfaction des joueurs vs critiques



🚀 Overview

Call of Data est un projet d’analyse visant à comprendre un problème clé de l’industrie du jeu vidéo :


🎯 Quels jeux reçoivent les meilleures évaluations et comment réduire l'écart entre les avis des joueurs et les critiques professionnelles ?

Ce projet aide un département PR (Relations Client) à améliorer la satisfaction utilisateur et à mieux gérer l’image des jeux.


🎯 Objectifs
- 📊 Comparer notes joueurs vs critiques
- 📉 Identifier les écarts de perception

🕹️ Analyser les tendances par :
 - année
 - plateforme
 - popularité

🚨 Détecter des phénomènes comme le review bombing


📊 Dataset

- 🎮 Jeux	16 719
- 📅 Période	1985 – 2016
- ⭐ Données	Notes critiques & joueurs
- 🕹️ Plateformes	Consoles + caractéristiques

🧠 Key Insights

✨ Ce que l’analyse révèle :

  - 📈 Depuis 2004, les critiques sont globalement plus positives que les joueurs
  - 💥 Certains jeux subissent des review bombings massifs
  - ⚠️ Les notes critiques créent parfois des attentes irréalistes
  - 🎯 L’expérience joueur ≠ perception des médias
